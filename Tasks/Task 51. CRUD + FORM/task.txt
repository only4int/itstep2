Имеется 3 поля ввода: 
    Название населенного пункта - текстовое поле ввода, 
    Район - выпадающий список,
    Тип населенного пункта - радиокнопки.
    
Также имеются 2 кнопки: Добавить и Сохранить. И еще есть пустая таблица следующего вида:
<table>
    <thead>
        <tr>
            <th>Название</th>
            <th>Раойн</th>
            <th>Тип</th>
            <th>Действия</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Витебск</td>
            <td>Витебский район</td>
            <td>Город</td>
            <td>
                <button class="edit-button">Редактировать</button>
                <button class="delete-button">Удалить</button>
            </td>
        </tr>
    </tbody>
</table> 

При нажатии кнопки "Добавить" в таблицу добавляется новая строка в соответсвии со структурой таблицы.
При нажатии кнопки "Удалить" из таблицы удаляется соотвествующая строка.
При нажатии кнопки "Редактировать" в поля заносятся данные из ячеек таблицы, а сама строка должна подсвечиваться фоном, а затем после нажатия кнопки "Сохранить" перезиписываются данные из полей в ту строку, которая редактируется. 

Все данные при добавлении и редактировании обязательно должны проверятся на заполнение.
